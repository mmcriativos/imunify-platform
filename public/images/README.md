# INSTRUÇÕES PARA ADICIONAR OS LOGOS

## Arquivos necessários:

### 1. Logo completo horizontal
**Arquivo:** `imunify-logo.png`
**Dimensões:** 180-250px de largura, 40-60px de altura
**Formato:** PNG com fundo transparente
**Usar:** O logo horizontal que você enviou (primeiro arquivo)

### 2. Ícone (apenas símbolo)
**Arquivo:** `imunify-icon.png`  
**Dimensões:** 512x512px (quadrado)
**Formato:** PNG com fundo transparente
**Usar:** Apenas o escudo com check (sem texto "Imunify")

## Como adicionar:

1. Salve o logo horizontal como: `public/images/imunify-logo.png`
2. Crie uma versão apenas do ícone e salve como: `public/images/imunify-icon.png`
3. Os arquivos já estão referenciados no código

## Onde os logos aparecem:

✅ Landing page - header (logo horizontal)
✅ Página de registro - header (logo horizontal)
✅ Favicon - tab do browser (ícone)
⏳ Dashboard - sidebar (aguardando implementação)

## Cores aplicadas:
- Cyan: #3ebddb
- Green: #77ca73
- Gradientes atualizados em toda aplicação
